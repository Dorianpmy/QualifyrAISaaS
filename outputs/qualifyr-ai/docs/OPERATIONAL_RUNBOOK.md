# Runbook opérationnel

1. Appliquer le schéma Supabase additif dans un environnement de test.
2. Vérifier les variables serveur Supabase ; ne jamais exposer la clé de service au navigateur.
3. Terminer l’onboarding, confirmer le pack et laisser Autopilot terminer ses étapes.
4. En cas d’échec, utiliser **Réessayer** : les étapes terminées sont conservées.
5. Ouvrir **Réglages entreprise**, corriger le premier point puis lancer **Tester mon système**.
6. Pour le support, copier uniquement le diagnostic anonymisé.

Ne pas présenter les workers, publications, réponses automatiques, SMS ou automatisations externes comme disponibles avant leur implémentation et leurs tests dédiés.
