# Centre de santé

Le centre de santé se trouve dans **Réglages entreprise**. Il vérifie l’espace, le pack, le pipeline CRM, le formulaire, le responsable, le mode sûr de l’assistant, les automatisations internes et Brain.

Les états sont : opérationnel, à compléter, bloquant et non disponible. Une seule prochaine action est mise en avant. Le diagnostic copiable passe par une liste blanche et exclut emails, téléphones, tokens, contenus clients et secrets.
